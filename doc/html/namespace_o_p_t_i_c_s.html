std::vector< DataVector > OPTICS::extract_clusters ( const DataVector &  result,
const std::vector< unsigned int > &  cluster_borders,
real  outlier_threshold 
)

Partitions the specified OPTICS ordered data points along the given cluster borders. Points that lie above a specified threshold are put into a separate outlier cluster.

Parameters
resultThe OPTICS ordered result vector of the optics function.
cluster_bordersA vector of indices specifiying the cluster borders. IMPORTANT: The vector must be sorted in ascending order.
outlier_thresholdAll values above that outlier_threshold are considered outliers and will be put in a special outlier cluster. Is the threshold value set to 0 or negative no point will be considered as an outlier.
Returns
A vector of different disjoint data point containers, each making up one cluster. The first container stores the data points that are considered outliers.
