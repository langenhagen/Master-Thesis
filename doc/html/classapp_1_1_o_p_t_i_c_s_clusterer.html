Given the OPTICS ordered output, finds all maxima peaks with a persistence greater than a given threshold. These are presumably cluster-borders.

Parameters
reachabilitiesThe OPTICS ordered reachability distances of the DataPoints that where the input of the OPTICS function.
persistenceThe persistence of the histogram peaks to retain.
Returns
All histogram peak indices that are above the given persistence threshold. The indices are ordered in ascending order to the persistence of the peaks at these positions.

Given the OPTICS ordered output, finds the k most persistent maxima peaks of the reachability distances, which are presumably cluster-borders.

Parameters
reachabilitiesThe OPTICS ordered reachability distances of the DataPoints that where the input of the OPTICS function.
n_clustersthe number of clusters that shall is we want to extract. The n_clusters-1 most persistent histogram peaks are assumed to be their borders. If less than n_clusters-1 histogram peaks exist, all peak incides will be returned.
Returns
All the n_clusters-1 biggest histogram peak indices . The indices are ordered in descending order to the persistence of the peaks at these positions.
