Detailed Description

Contains the whole stuff needed for processing an image. It chooses the appropriate saliency detector and feature extractor and contains the processing chain within one function call. It also handles the writing of intermediate results to disk. The object maintains (1) The saliency map generator. (2) Creates salient object FeatureExtractor.
