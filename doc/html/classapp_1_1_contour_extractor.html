uint  n_frequencies 
) const
inlineprotected

Creates a normalized fourier descriptor from a given contour. The descriptor will be made translation invariant, scale-invariant. Furthermore, only the magnitude of the fourier description will be kept, since the phase in not rotation invariant. See AIA Lecture 2 or AIA Homework 2 for details.

Parameters
